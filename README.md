# GUI Calculator

 Description
This is a simple GUI-based Calculator application that performs basic arithmetic operations like addition, subtraction, multiplication, and division.

 Features
- User-friendly interface
- Basic operations (+, -, *, /)
- Real-time calculation
- Clear button functionality

 Technologies Used
- Python (Tkinter) / Java (Swing/JavaFX)

 How to Run
1. Download or clone the repository
2. Open the project folder
3. Run the main file:
   - Python: `python calculator.py`
   - Java: compile and run the main class
